# Peace Inu Crypto-Inspired Website

## How to use

1. Place a photo of your Shiba Inu named Peace in this folder and name it `peace.jpg`.
2. Open `index.html` in your browser to view the site.

- The site is responsive, dark-themed, and features animated glowing effects.
- You can customize `styles.css` and `script.js` for more effects or branding.

Enjoy your futuristic, crypto-inspired site for Peace!
